This directory contains a Strict SysY testcase converted from UVa1103 "Ancient Messages".

- `testfile.txt`: Strict SysY program (no headers, `getint()` input, `printf` output).
- `in.txt`: input data, **one integer per line**.
  - Format: `H`, `W`, then `H*W` integers in `[0,15]` (hex digits per row, left to right), then sentinel `0 0`.
- `ans.txt`: expected output generated by the provided runner.
